# E-Commerce Website (Amazon-Clone)
Building a Amazon-clone website using React-JS

## Concepts used in the project
1. React Hooks
    - useContext
    - useReducer
2. React Router
3. Material UI
4. Hgher Order Component
5. React-currency-format
